# 안녕하세요 👋

이 글은 GitHub 저장소에서 **자동으로 Blogger 에 발행**된 첫 글입니다.

작성 흐름은 이렇습니다.

1. `posts/` 폴더에 마크다운으로 글을 쓰고
2. `git push` 하면
3. GitHub Actions 가 Blogger API 로 발행합니다.

> `draft: true` 라서 지금은 **초안**으로만 올라갑니다.
> 정식 발행하려면 `draft: false` 로 바꾸고 다시 push 하세요.
